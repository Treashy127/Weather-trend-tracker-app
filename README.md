Weather Trend App 🌦️

An interactive web app that visualizes historical and current weather trends. Track temperature, precipitation, and other climate metrics over time for any location, all displayed in sleek, interactive charts.

Features
Display historical and current weather trends
Interactive charts using Recharts
Responsive design for desktop and mobile
Search for any city or location
Fetches real-time data from a weather API
Tech Stack
React.js – Frontend framework
CSS – Styling and layout
Recharts – Interactive charts for visualizing trends
Weather API – Real-time and historical weather data
Installation

Clone the repository:

git clone https://github.com/your-username/weather-trend-app.git

Navigate to the project folder:

cd weather-trend-app

Install dependencies:

npm install

Start the development server:

npm start
Usage
Enter a city or location in the search bar
View temperature, precipitation, and other trends over time
Interact with charts to explore data visually
